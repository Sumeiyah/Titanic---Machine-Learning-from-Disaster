# Titanic Survival Prediction

This project explores the Titanic dataset from Kaggle and applies machine learning techniques to predict passenger survival.

## Project Workflow
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Logistic Regression
- Random Forest Classification
- Model evaluation using:
  - Accuracy
  - Confusion Matrix
  - Precision
  - Recall
  - F1-score

## Key Findings
- Female passengers had higher survival rates
- First-class passengers were more likely to survive
- Fare was one of the most important predictive features
